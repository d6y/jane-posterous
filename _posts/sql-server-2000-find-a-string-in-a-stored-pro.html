title: SQL Server 2000 - find a string in a stored procedure date: 2006/07/20 03:28:00 -0700 tags: [{"name":"code","id":58678584},{"name":"SQLServer","id":3773152}] author: Jane Dallaway alias: /sql-server-2000-find-a-string-in-a-stored-pro I've been debugging and unit testing some involved stored procedures, and I've left a trail of PRINT statements all over the place. I was having real difficulties tracking the last 2 down, so I wrote the following SQL to help me track them down.

SELECT o.name
FROM syscomments c
INNER JOIN sysobjects o
ON c.id = o.id
WHERE o.xtype = 'P'
AND text LIKE 'PRINT @'
