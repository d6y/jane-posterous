title: SQL Server Integration Services date: 2006/08/14 04:09:00 -0700 tags: [{"name":"SQLServer","id":58673819}] author: Jane Dallaway alias: /sql-server-integration-services I've been doing some work with Integration Services in the past few weeks. Some resources have proved to be a bit too much like hard work, others have been more useful.

MSDN has a series of tutorials

Blogs:
There are some good blogs kicking around - these are the ones that I've had a few hints and tips from and that have been added to my RSS aggregator:

  • Kirk Haselden
  • Jamie Thomson - especially the SSIS: Suggested Best Practices and naming conventions
  • Daniel Read who amongst his other helpful posts has put together an index to the excellent SQL 2005 SSIS How to series by Marcin Policht at Database Journal.

Forums:
  • MSDN SQL Server Integration Services

Webcasts:
  • Business Intelligence with SQL Server 2005: Transform Data into Decisions

Books:
  • Professional SQL Server 2005 Integration Services is a bit basic and doesn't really go into any of the more complex bits and pieces. An alright introduction, but to be honest some of the web sites mentioned above can give you that level of information.

  • Microsoft SQL Server 2005 Integration Services. On order, not released until end of August 2006.

Update: After signing up for one of the webcasts the other day, a useful email came through pointing me at MSDN Integration Services which has an impressive set of links.
Futher Update: Thanks to Linda for pointing out via Daniel Read's index that there is an index to the SSIS articles on Database Journal.
