title: WebDD Overview date: 2007/02/03 14:37:00 -0800 tags: [{"name":"event","id":58674410}] author: Jane Dallaway alias: /webdd-overview I had a great day at WebDD today and attended some good sessions - which I'll go into more later. The Scott Guthrie sessions were hugely over-subscribed, but were recorded so I hope to get a chance to see them at a later stage. Initally not a lot of swag, but I got to leave with a copy of Expression Web as well as a pen and some magazines.

The sessions I attended today were:

  • Expression Web - an excellent introduction to a tool which looks great

  • Doing it in style: creating beautiful sites, the web standards way - a bit too much content for the hour, so I'm hoping to see the slides to review the bits that were skipped over

  • Quick and dirty Usability tests - one week, no budget, and no usability facility - an interesting overview of usability testing - something I've never experienced

  • Ruby On Rails for .NET Developers - excellent presentation, great looking tool and framework

  • Unobtrusive Javascript with Rails - graceful degradation of javascript

And so there ends my first ever conference, I'm not sure why but I've never managed to attend one before. I'm definitely hoping to attend the next WebDD event, and also probably the DeveloperDeveloperDeveloper! events too.
